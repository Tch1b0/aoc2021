# aoc2021

## Taskfile commands

- `build` Builds the aoc-helper

## aoc-helper
Usage: `aoc [arguments]`
-    `-h`, `--help`                       Show help
-    `-l [LANG]`, `--lang=[LANG]`         Set the language you want to use
-    `-d [DAY]`, `--day=[DAY]`            Set the day you want to work on
-    `-nt`, `--no-template`               Don't generate a template

## Requierments for the aoc-helper

- Crystal
- Taskfile
